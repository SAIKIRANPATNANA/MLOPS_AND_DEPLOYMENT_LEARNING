# MLOps and Deployment Learning

A curated learning repository documenting my hands-on journey through **MLOps, deployment workflows, reproducibility, packaging, and production-oriented machine learning practices**.

This repository brings together the material I have used to understand how machine learning systems move beyond notebooks into more structured, maintainable, and deployable workflows. Rather than focusing only on model building, this learning track focuses on the surrounding engineering practices that make ML systems usable in real environments.

## What This Repository Covers

- **MLOps foundations** through project structure, reproducibility, and workflow organization
- **Deployment-oriented learning** through practical packaging and application-serving concepts
- **Experiment-to-system thinking** by connecting model development with serving and operations
- **Containerization practice** through Docker-based learning and environment setup

## Learning Tracks Inside This Repo

### MLOps Learning Track
The `MLOPS_LEARNING` section focuses on concepts and hands-on work around production-minded machine learning, including:
- project structuring for ML systems
- workflow standardization
- experiment organization
- reproducibility and maintainability
- practical learning from end-to-end ML examples

Representative learning material includes:
- `End-to-end-Machine-Learning-Project-with-MLflow`
- `MlopsByFreeCodeCamp`
- `MlopsByIneuronLearning`

### Docker and Deployment Learning Track
The `DOCKER_LEARNING` section focuses on deployment-adjacent engineering basics, including:
- containerization fundamentals
- packaging applications for consistent execution
- understanding how deployment environments differ from local development
- supporting application portability
